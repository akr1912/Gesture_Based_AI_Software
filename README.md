1. Volume Up/Down
2. Brigthness High/Low
3. Scroll Up/ Scroll Down
4. Mouse Left Click / Right Click
5. Full Mouse Movement Using Index Finger

2. Ask the time 
3. Check the weather
4. Google anything by giving the voice commands
5. Open Youtube
6. Open Google

9. Shutdown the system
